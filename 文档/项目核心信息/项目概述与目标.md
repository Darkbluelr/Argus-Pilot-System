# 项目概述与目标

## 1. 项目名称

Argus Pilot System

## 2. 项目目标

构建一个模块化、可扩展的通用计算机控制 (GCC) 框架，旨在通过引入动态知识图谱 (DKG)、标准化的适配器接口和增强的认知能力，实现对不同桌面和 Web 应用程序的精细化、智能化控制。

本项目旨在超越现有框架（如 CRADLE）的局限，提供一个更通用、更强大、更易于扩展的基础设施，使 AI Agent 能够更深入地理解应用状态、更有效地规划和执行任务，并最终打破 AI 在复杂计算机交互场景下的应用边界。
